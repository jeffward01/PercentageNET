Simple helpers to count percentage and more.

`Install-Package PercentageNET -Version 1.0.0`

## Documentation
`.Count(1000, 20)` = 200

`.Substract(1000, 20)` = 800

// simple mode
`.Add(1000, 20)` = 1200

// upcharge mode
`.Add(1000, 20, AddMode.Upcharge)` = 1250
